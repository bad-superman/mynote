    Rueidis е®ўжҲ·з«ҜиҝһжҺҘ Redis Cluster й—®йўҳз¬”и®°

    й—®йўҳиғҢжҷҜ

    еңЁдҪҝз”Ё rueidis е®ўжҲ·з«ҜиҝһжҺҘ Redis Cluster ж—¶йҒҮеҲ° NOAUTH Authentication required й”ҷиҜҜпјҢеҚідҪҝй…ҚзҪ®дәҶжӯЈзЎ®зҡ„еҜҶз ҒгҖӮ

    йҒҮеҲ°зҡ„й—®йўҳ

    й—®йўҳ 1: NOAUTH Authentication required

    зҺ°иұЎпјҡй…ҚзҪ®дәҶеҜҶз ҒдҪҶиҝһжҺҘ Redis Cluster ж—¶д»Қз„¶жҠҘи®ӨиҜҒй”ҷиҜҜ

    еҺҹеӣ пјҡrueidis еңЁ Cluster жЁЎејҸдёӢдјҡе°қиҜ•иҮӘеҠЁжЈҖжөӢйӣҶзҫӨжӢ“жү‘пјҢдҪҶеңЁжҹҗдәӣ Redis Cluster й…ҚзҪ®дёӢпјҢи®ӨиҜҒдҝЎжҒҜеҸҜиғҪж— жі•жӯЈзЎ®дј йҖ’еҲ°жүҖжңүиҠӮзӮ№гҖӮ

    и§ЈеҶіж–№жЎҲпјҡдҪҝз”Ё ForceSingleClient: true ејәеҲ¶дҪҝз”ЁеҚ•жңәе®ўжҲ·з«ҜжЁЎејҸгҖӮ

    client, err := rueidis.NewClient(rueidis.ClientOption{
    InitAddress:       []string{"10.213.64.8:6379"},
    Password:          "your-password",
    ForceSingleClient: true,
    DisableCache:      true,  // еҝ…йЎ»еҗҢж—¶и®ҫзҪ®
})

    й—®йўҳ 2: CLIENT TRACKING ON OPTIN й”ҷиҜҜ

    й”ҷиҜҜдҝЎжҒҜпјҡ

    ClientOption.DisableCache must be true for redis not supporting client-side caching or not supporting RESP3

    еҺҹеӣ пјҡ

    • rueidis й»ҳи®ӨеҗҜз”Ёе®ўжҲ·з«Ҝзј“еӯҳпјҲclient-side cachingпјүеҠҹиғҪ

    • иҜҘеҠҹиғҪйңҖиҰҒ Redis ж”ҜжҢҒ RESP3 еҚҸи®®е’Ң CLIENT TRACKING е‘Ҫд»Ө

    • еҰӮжһң Redis дёҚж”ҜжҢҒиҝҷдәӣзү№жҖ§пјҢеҝ…йЎ»и®ҫзҪ® DisableCache: true

    з»“и®ә

    дёҠйқўжҳҜaiеҲҶжһҗзҡ„з»“жһ„пјҢзңҹжӯЈзҡ„й—®йўҳжҳҜRueidis v1.0.54 жңүbug, еңЁеҸ‘йҖҒHELLOе‘Ҫд»Өд№ӢеүҚжІЎжңүAUTHпјҢжӣҙж–°RueidisзүҲжң¬е°ұеҸҜд»ҘдәҶз¬‘и„ёрҹҳҒ

    RESP3 еҚҸи®®

    д»Җд№ҲжҳҜ RESP3

    RESP (Redis Serialization Protocol) жҳҜ Redis зҡ„е®ўжҲ·з«Ҝ - жңҚеҠЎеҷЁйҖҡдҝЎеҚҸи®®гҖӮ

    зүҲжң¬ еҸ‘еёғе№ҙд»Ҫ дё»иҰҒзү№жҖ§
    RESP2 2012 еҹәзЎҖж•°жҚ®зұ»еһӢпјҲеӯ—з¬ҰдёІгҖҒж•°з»„гҖҒж•ҙж•°гҖҒй”ҷиҜҜгҖҒз©әпјү
    RESP3 2018 ж–°еўһж•°жҚ®зұ»еһӢгҖҒе®ўжҲ·з«Ҝзј“еӯҳж”ҜжҢҒгҖҒжӣҙеҘҪзҡ„й”ҷиҜҜеӨ„зҗҶ

    RESP3 ж–°еўһзү№жҖ§

    1. ж–°зҡ„ж•°жҚ®зұ»еһӢпјҡ

      • Push ж¶ҲжҒҜпјҲжңҚеҠЎеҷЁдё»еҠЁжҺЁйҖҒпјү

      • Map зұ»еһӢпјҲеҺҹдёәж•°з»„пјү

      • Set зұ»еһӢ

      • Double зұ»еһӢ

      • Boolean зұ»еһӢ

    2. е®ўжҲ·з«Ҝзј“еӯҳж”ҜжҢҒпјҡ

      • CLIENT TRACKING е‘Ҫд»Ө

      • жңҚеҠЎеҷЁеҸҜд»ҘйҖҡзҹҘе®ўжҲ·з«Ҝзј“еӯҳеӨұж•Ҳ

    3. жӣҙеҘҪзҡ„й”ҷиҜҜеӨ„зҗҶпјҡ

      • жӣҙиҜҰз»Ҷзҡ„й”ҷиҜҜдҝЎжҒҜ

      • й”ҷиҜҜеҲҶзұ»

    rueidis дёҺ RESP3

    rueidis й»ҳи®Өе°қиҜ•дҪҝз”Ё RESP3пјҡ

    • еҰӮжһң Redis ж”ҜжҢҒ RESP3пјҢеҲҷдҪҝз”Ё RESP3

    • еҗҰеҲҷйҷҚзә§еҲ° RESP2

    еҸҜд»ҘйҖҡиҝҮ AlwaysRESP2: true ејәеҲ¶дҪҝз”Ё RESP2гҖӮ

    Client-Side Caching (е®ўжҲ·з«Ҝзј“еӯҳ)

    д»Җд№ҲжҳҜе®ўжҲ·з«Ҝзј“еӯҳ

    Redis е®ўжҲ·з«Ҝзј“еӯҳжҳҜдёҖз§ҚдјҳеҢ–жңәеҲ¶пјҢе…Ғи®ёе®ўжҲ·з«ҜеңЁжң¬ең°зј“еӯҳд»ҺжңҚеҠЎеҷЁиҜ»еҸ–зҡ„ж•°жҚ®пјҢеҗҢж—¶жңҚеҠЎеҷЁдјҡи·ҹиёӘе®ўжҲ·з«Ҝзҡ„зј“еӯҳе№¶еңЁж•°жҚ®еҸҳжӣҙж—¶йҖҡзҹҘе®ўжҲ·з«ҜеӨұж•ҲгҖӮ

    еҗҜз”ЁжқЎд»¶

    е®ўжҲ·з«Ҝзј“еӯҳйңҖиҰҒпјҡ

    1. Redis 6.0+ (еј•е…Ҙ CLIENT TRACKING е‘Ҫд»Ө)

    2. RESP3 еҚҸи®® (йңҖиҰҒ Push ж¶ҲжҒҜж”ҜжҢҒ)

    3. rueidis зҡ„ DisableCache: false (й»ҳи®Ө)

    дёәд»Җд№ҲйңҖиҰҒ DisableCache: true

    еҪ“ Redis дёҚж”ҜжҢҒ RESP3 жҲ– CLIENT TRACKING ж—¶пјҡ

    • rueidis ж— жі•еҗҜз”Ёе®ўжҲ·з«Ҝзј“еӯҳ

    • дҪҶ ForceSingleClient жЁЎејҸдёӢд»Қе°қиҜ•дҪҝз”Ёзј“еӯҳеҠҹиғҪ

    • еҜјиҮҙй”ҷиҜҜпјҡCLIENT TRACKING ON OPTIN

    и§ЈеҶіж–№жЎҲпјҡи®ҫзҪ® DisableCache: true зҰҒз”Ёе®ўжҲ·з«Ҝзј“еӯҳеҠҹиғҪгҖӮ

    жҖ§иғҪеҪұе“Қ

    зҰҒз”Ёе®ўжҲ·з«Ҝзј“еӯҳзҡ„еҪұе“Қпјҡ

    • еўһеҠ зҪ‘з»ңиҜ·жұӮпјҡжҜҸж¬ЎиҜ»еҸ–йғҪйңҖиҰҒи®ҝй—® Redis

    • еўһеҠ е»¶иҝҹпјҡж— жі•еҲ©з”Ёжң¬ең°зј“еӯҳ

    жЈҖжҹҘжё…еҚ•

    • [ ] Redis зүҲжң¬жҳҜеҗҰ >= 6.0 (ж”ҜжҢҒ CLIENT TRACKING)

    • [ ] Redis жҳҜеҗҰж”ҜжҢҒ RESP3 (redis-cli HELLO жЈҖжҹҘ proto)

    • [ ] зҪ‘з»ңжҳҜеҗҰе…Ғи®ё CLUSTER SLOTS е‘Ҫд»Ө

    • [ ] жҳҜеҗҰйңҖиҰҒеҲҶеёғејҸжҖ§иғҪпјҲеҰӮжһңдёҚйңҖиҰҒпјҢForceSingleClient жҳҜе®үе…Ёзҡ„йҖүжӢ©пјү
