# 一、核心风控计算因子（Risk Factors）

## 1️⃣ 资产与账户类因子（Account / Asset Factors）

| 因子  | 英文  | 说明  |
| --- | --- | --- |
| 现金  | Cash | 含 Dividend、Interest |
| 在途资金 | Cash in Transit | 已入账未清算资金 |
| 货币基金 / 钱包 | MMF / Wallet | 可变现资产 |
| 持仓市值 | MV / Gross Position Value | 市价单按最新价\*1.05 |
| 期权市值 | Option Market Value | 需单独扣除 |
| 信用额度 | Credit Line | 非 margin loan 的额外信用 |
| 初始保证金 | IM (Initial Margin) | 开仓锁定 |
| 维持保证金 | MM (Maintenance Margin) | 强平线 |
| 已锁定金额 | Locked EE | 手续费、利息、税费、已申请出金、IPO 等 |
| 融券卖出所得 | Short Proceeds | 做空锁仓资金 |

---

## 2️⃣ 派生风控指标（Derived Risk Metrics）

| 指标  | 英文  | 风控用途 |
| --- | --- | --- |
| 净清算价值 | NLV (Net Liquidation Value) | 账户真实净值 |
| 含借权益 | ELV (Equity with Loan Value) | 风控核心基准 |
| 剩余资产 | EE (Excess Equity) | 开仓检查 |
| 可用剩余资产 | AEE (Available EE) | 扣锁定项后 |
| 剩余流动性 | EL (Excess Liquidity) | 强平判断 |
| 风险率 | RR (Risk Ratio) | 预警 & 通知 |
| Post EE / Post AEE |     | 期权行权模拟后指标 |

---

## 3️⃣ 风控参数类因子（Risk Parameters）

| 参数  | 说明  |
| --- | --- |
| IM Rate | 初始保证金比例 |
| MM Rate | 维持保证金比例 |
| Greeks | Delta / Gamma / Theta / Vega |
| 波动率 | Volatility |
| 集中度阈值 | 单标的、行业、发行人 |
| 手数限制 | 单笔 / 日内 |
| 成交量占比 | 不超过历史均量 25% |
