在 Windows 系统中，设置开机启动和定时任务主要依赖两个非常实用的管理工具：**任务管理器**和**任务计划程序**。前者用于管理登录时自动运行的程序，后者则用于安排脚本或程序在特定时间或事件下自动执行。

---

### 🚀 一、设置开机启动项

设置开机启动项，目的是让指定的程序在每次你登录电脑时自动运行。这里有几种常用的方法。

#### 方法一：通过任务管理器（最常用）

这是管理已安装应用的启动状态最直接的方法。

1.  **打开任务管理器**：按下快捷键 `Ctrl + Shift + Esc`。
    
2.  **切换到“启动”选项卡**：在任务管理器窗口中，点击上方的“启动”标签页。
    
3.  **启用或禁用启动项**：在列表中，右键点击你想要**启用**（设置为开机启动）或**禁用**的程序，在弹出的菜单中选择“启用”或“禁用”即可。
    

#### 方法二：通过启动文件夹（最灵活）

如果你需要启动的程序不在任务管理器的列表中，或者你想运行一个脚本，可以把它放到启动文件夹里。

1.  **打开“启动”文件夹**：按下 `Win + R` 键，在运行框中输入 `shell:startup`，然后点击“确定”。
    
2.  **添加快捷方式**：将你想要开机自动运行的程序或脚本的**快捷方式**，直接复制粘贴到这个文件夹中。下次登录时，它就会自动启动。
    

#### 方法三：通过系统设置（适用于Windows 11）

1.  打开 **“设置”** 应用，点击左侧的 **“应用”**。
    
2.  选择 **“启动”**。
    
3.  在这里，你可以看到应用列表，通过开关来控制它们是否在登录时启动。
    

> **一点提示**：过多的开机启动项会拖慢电脑的启动速度，建议只保留最必要的项目。

### ⏰ 二、设置定时任务

定时任务的核心工具是 **“任务计划程序”**，它就像一个系统闹钟，可以让你精确地安排程序或脚本的执行时间。

#### 方法一：使用图形界面（任务计划程序）

这种方式最为直观，适合大多数场景。

1.  **打开任务计划程序**：
    
    -   在Windows搜索框中输入“**任务计划程序**”并打开。
        
    -   或者，按下 `Win + R`，输入 `taskschd.msc` 并回车。
        
2.  **创建基本任务**：在窗口右侧的“操作”栏中，点击 **“创建基本任务”**。这会启动一个向导，引导你完成设置。
    
3.  **按照向导设置**：
    
    -   **名称和描述**：给你的任务起一个容易识别的名字。
        
    -   **触发器**：选择任务何时开始。可以是每天、每周、每月，也可以是**计算机启动时**或**用户登录时**。
        
    -   **操作**：选择“启动程序”，然后浏览并选中你想要执行的程序或脚本。
        
    -   **完成**：检查设置摘要，点击“完成”即可。
        

#### 方法二：使用命令行（`schtasks`）

如果你需要批量创建任务或进行远程管理，可以使用强大的 `schtasks` 命令。以下是一些常用示例：

| 任务场景 | 命令示例 | 说明  |
| --- | --- | --- |
| **每天固定时间运行** | `schtasks /create /tn "任务名称" /tr "C:\路径\脚本.bat" /sc daily /st 09:00` | 每天上午9点运行脚本。 |
| **每次开机运行** | `schtasks /create /tn "开机任务" /tr "C:\路径\程序.exe" /sc onstart` | 计算机启动时自动运行。 |
| **每次登录时运行** | `schtasks /create /tn "登录任务" /tr "C:\路径\监控.exe" /sc onlogon` | 任意用户登录计算机时运行。 |
| **每隔一段时间运行** | `schtasks /create /tn "频繁任务" /tr "C:\路径\检查.bat" /sc minute /mo 15` | 每15分钟运行一次。 |
| **管理任务** | `schtasks /query` (列出任务)<br>`schtasks /run /tn "任务名称"` (立即运行)<br>`schtasks /change /tn "任务名称" /DISABLE` (禁用)<br>`schtasks /delete /tn "任务名称" /f` (删除，/f表示强制) | 查看、立即执行、禁用和删除计划任务。 |

### 💡 总结一下

| 功能  | 核心工具 | 常用入口 |
| --- | --- | --- |
| **开机启动** | 任务管理器 | `Ctrl + Shift + Esc` -> 启动 |
| **开机启动（高级）** | 启动文件夹 | `Win + R` -> `shell:startup` |
| **定时任务** | 任务计划程序 | 开始菜单搜索“任务计划程序” |
